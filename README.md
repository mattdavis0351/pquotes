# Tech Quotes

A tech inspiration generator for your terminal

### Usage

#### Installing

1. Install [Go](https://golang.org/)
1. Clone this repo
1. Change working dir to be `/cmd/tech-quote/`
1. From terminal run `go install`

#### Using

1. Open a terminal
1. Type `tech-quote help` or `tech-quote random`
